ABOUT THIS WEBSITE

This website leverages the power of Deep Learning by using Convolutional Neural Networks (CNNs) to classify a given image into one of 8 skin diseases. I decided to learn CNNs by actual implementation, so I stumbled upon this dataset on Kaggle and thought to create a CNN and implement it using Flask.

These skin diseases include:

Cellulitis
Impetigo
Athlete-Foot
Nail-Fungus
Ringworm
Cutaneous-larva-migrans
Chickenpox
Shingles
